# Emacs

This configuration fits in less than 1000 lines of Emacs Lisp, including comments and whitespaces.

It has performance and simplicity in mind, using most of Emacs builtin features. The configuration started as a template from [rexim](https://github.com/rexim/dotfiles) dotfiles, and inspired by [minimal-emacs.d](https://github.com/jamescherti/minimal-emacs.d), [prot](https://github.com/protesilaos/dotfiles) and [emacs-solo](https://github.com/LionyxML/emacs-solo).
